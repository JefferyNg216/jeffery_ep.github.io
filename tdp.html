The Data Professional

From The Data Professional module, we have been introduced four what initially to start our learning:-

  • What is data science and how is it working?
  • What is the role and responsibilities of a data scientist?
  • What is the future and career path as a data scientist?
  • What is the challenge and advantage of being a data scientist?

    • And during the module, we have gained below key knowledges:

  • Data security and protection by utilise GDPR, ISMS and ISO 27000 / 27001
  • Personal data security
  • Data management lifecycle
  • Master Data Management
  • Programming skill of Python and SQL 
  • Data manipulation
  • Database structure, models, relational design, social impacts and security
  • Couple of algorithms framework (e.g. CALGO)
  • Design and Analysis of Algorithms
  • Data handling process, Big data acquisition and data analysing
  • Data Discretisation, Normalisation and Standardisation
  • APIs functionalities
  • Real world data issue (e.g. Incomplete, Noisy and Inconsistent)
  • Technique of handling missing data and its pros and cons (Listwise deletion,  Pairwise deletion and Variable Deletion)
  • Encoding methodology (Label Encoding and OneHotEncoder)
  • Data measurement (Qualitative, Quantitative and  The four scales of measurement - Nominal, Ordinal, Interval and Ratio scale measurement)
  • IoT fundamental
  • Data visualisation (Chart type and usage)
  • Storytelling with Data
  • Business Analysis
  • Blockchain

    • Importantly, from the module assignment Data Analytics Report and Data Analytics Implementation, we learnt how the design is essential to the data analytics implementation which could outline and review the objectives that meet the requirement. Finally, during the Data Analytics Implementation, we adopt what we learnt from the module, especially by using python to implement data pre-processing, then visualise the result with the viewpoint to explain the whole analysing aspect.
